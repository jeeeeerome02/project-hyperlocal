# UI/UX Wireframe Descriptions

## Design System Foundation

### Visual Language
- **Primary Color**: `#FF6B35` (Warm orange â€” energy, community, warmth)
- **Secondary Color**: `#0984E3` (Trust blue â€” safety, reliability)
- **Background**: `#FAFAFA` (off-white), Dark mode: `#1A1A2E`
- **Text**: `#2D3436` primary, `#636E72` secondary
- **Font**: Inter (UI), system fallback stack
- **Border Radius**: 12px cards, 8px buttons, 24px pills
- **Shadows**: Subtle (`0 2px 8px rgba(0,0,0,0.08)`) â€” avoid heavy depth
- **Map Style**: Mapbox Light (custom), muted base to let pins pop

### Design Principles
1. **Map is King**: The map occupies 70%+ of viewport at all times
2. **One-Thumb Reach**: All primary actions reachable with one thumb on mobile
3. **Glanceable Pins**: Category is identifiable from icon + color at any zoom
4. **Progressive Disclosure**: Tap for details; don't overload the map
5. **Ephemeral Feel**: UI hints at impermanence (countdown timers, fading pins)
